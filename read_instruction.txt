Для архивирования:
1. Выбираем файл из меню "Файл/Выбрать файл для архивирования" или вводи в большое окно необходимый текст
2. Задаём имя файла, в котором сохраниться архивированный файл
3. Нажимаем кнопку архивировать
4. Выбираем в появившимся окне папку, куда сохраним файл
5. Архивация прошла успешно!!!!

Для распаковки:
1. Задаём имя файла для сохранения после распаковки
2. Нажимаем кнопку распоковать
3. Выбираем файл для распаковки
4. Результат распаковки распаковки отобразиться в окне
5. Чтобы сохранить файл выбирите "Файл/Сохранить файл после распаковки"

Примечание.
Если перед архивацией вы хотите сохранить введёный текст выберете также "Файл/Сохранить файл после распаковки". 
В данном случаи просто не обращайте внимание на название "Сохранить файл после распаковки". Ваш файл сохранить
в виде текстового файла в формате .txt, но не забывайте задавать имя файла к окне для его ввода.
